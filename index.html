About Me

I'm a technical writer and a mom, with two children, Amelia and Patrick. I've been writing docs that help web developers or about 5 years now. And I've written lots of simple samples, but there's tons of gaps in my knowledge. Last year, I decided to start building more complicated web applications, from start to finish. I discovered I like writing web applications, a lot more than I like to write docs about writing web applications. So I'm in this bootcamp because I want to write code intensely for 6 months. After that, I am hoping to transition, slowly, into an engineering role, fingers crossed, as a developer advocate in my existing team.
